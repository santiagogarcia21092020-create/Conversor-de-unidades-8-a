# Conversor de Unidades

Un conversor de unidades sencillo hecho en Python.

## ¿Qué hace este programa?

Este programa convierte diferentes tipos de unidades:

- Longitud
- Peso
- Temperatura
- Volumen

## Unidades soportadas

### Longitud
- m
- km
- cm
- mm
- mi
- ft
- in

### Peso
- kg
- g
- mg
- lb
- oz

### Temperatura
- c
- f
- k

### Volumen
- l
- ml
- m3
- gal
- cup

## Cómo ejecutar el programa

1. Asegúrate de tener Python 3 instalado
2. Abre la carpeta del proyecto
3. Ejecuta el programa

Ejemplo
=== Conversor de Unidades ===
1. Longitud
2. Peso
3. Temperatura
4. Volumen

Elige una opción (1-4): 1
Ingresa el valor: 10
Unidad de origen: km
Unidad destino: m

Resultado: 10000
